Siapa Saya?

Perkenalkan nama saya adalah Paulus Lestyo Adhiatma. Sebut saja saya seorang programmer Lulusan dari SMK N 2 Surakarta yang saat ini sedang menjalani perkuliahan di Universitas Sebelas Maret Semester 2. Saya memiliki pengalaman di magang menjadi seorang Web Developer di Duxeos Software House dan juga di kampus tercinta UNS di bagain Lembaga Penelitian dan Pengabdian Kepada Masyarakat (LPPM). Walaupun begitu saya masih sangat membutuhkan ilmu pengetahuan tetang programming baik itu dibidang bahasa pemrograman maupun wawasan tentang pemrograman.
